# Mini-FFB-Base
A mini FFB base using 4310 motor
<img src="img/draft 2024-04-05 190442.png" >
使用两个海泰4310电机，电机内置行星减速器；
电机驱动使用灯哥odrive双电机驱动板；
主控使用OpenFFBoard。
